# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

The HugeRTE Blazor component 1.0.0 is based on the HugeRTE Blazor component 2.2.1

## 1.0.0 - 2026-02-14

### Removed
- Removed `Readonly` property.
- Removed `CloudChannel` property.
- Removed `LicenseKey` property.

### Changed
- The `Disabled` property now toggles the `readonly` mode.

### Added
- New `CdnVersion` property.
