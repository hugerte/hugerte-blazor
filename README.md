# HugeRTE Blazor component

## About

This package is a thin wrapper around [HugeRTE](https://github.com/hugerte/hugerte) to make it easier to use in a Blazor application.

Documentation for this package is available at the [HugeRTE Blazor Integration docs page](https://github.com/hugerte/hugerte-docs/blob/main/integrations/blazor.md).
